# Modern CSS Mastery: The Ultimate Roadmap

Welcome to your structured, step-by-step journey to mastering CSS! This course is designed to take you from absolute beginner to senior-level frontend developer capable of building complex layouts, optimizing performance, and building accessible UI components.

## How to use this guide

1. **Follow Sequentially**: Each topic builds upon the previous one. Don't skip the fundamentals!
2. **Practice Daily**: CSS is best learned by doing. Create a `.html` and `.css` file next to each markdown file and run the examples.
3. **Build the Mini-Projects**: The real-world projects at the end of sections are crucial for cementing your knowledge.

## Daily Learning Plan

- **Days 1-3**: Phase 1 - Fundamentals (Selectors, Box Model, Units, Typography)
- **Days 4-7**: Phase 2 - Advanced Layouts (Flexbox, Grid, Media Queries)
- **Days 8-10**: Phase 3 - Advanced Styling & Architecture
- **Days 11-13**: Phase 4 - Animations, Internals & Performance
- **Days 14-16**: Phase 5 - Real-world Projects & Interview Prep

## Curriculum Overview

### Phase 1: Fundamentals
- `01-basics.md` - What is CSS, adding CSS, cascading, basic syntax
- `02-selectors.md` - Basic to advanced selectors (attribute, combinators)
- `03-specificity.md` - Understanding the "CASCADING" part, calculating specificity
- `04-box-model.md` - The most important concept: margin, border, padding, content
- `05-units.md` - Absolute vs Relative (px, rem, em, %, vw, vh)
- `06-typography-colors.md` - Fonts, colors, standardizing aesthetics

### Phase 2: Layouts & Responsive Design
- `01-display-positioning.md` - Block/Inline, relative, absolute, fixed, sticky
- `02-flexbox.md` - 1D layout deep dive
- `03-css-grid.md` - 2D layout deep dive
- `04-responsive-media-queries.md` - Mobile-first design, fluid typography

### Phase 3: Styling, Advanced Selectors & Architecture
- `01-shadows-borders-gradients.md` - Box-shadow, border-radius, linear/radial gradients
- `02-pseudo-classes-elements.md` - `:hover`, `:nth-child`, `::before`, `::after`

- `05-css-variables.md` - Custom properties for theming
- `06-architecture-bem-tailwind.md` - BEM, SMACSS, Utility-first logic

### Phase 4: Animations, Internals & Performance
- `01-transitions-animations-transforms.md` - Keyframes, 2D/3D transforms, hardware acceleration
- `02-internals-layout-rendering.md` - Critical rendering path, repaint vs reflow
- `03-performance-optimization.md` - Reducing CSS payload, containing repaints, containment
- `04-accessibility-compatibility.md` - Prefixes, functional CSS for screen readers, `prefers-reduced-motion`

### Phase 5: Real-world, Projects & Interviews
- `01-ui-components.md` - Navbars, Cards, Modals, Dashboards
- `02-layout-challenges.md` - Holy Grail layout, Masonry, Centering everything
- `03-interview-questions.md` - Tricky scenarios and common senior interview questions
